# smartphone-chatbot
AI shopping chat agent

1. First, set-up .env file:<br>

<ul>
<li>GOOGLE_API_KEY={google-api-key}
<li>TECHSPEC_API_ID={techspec-api-id}
<li>TECHSPEC_API_KEY={techspec-api-key}
</ul>


2. install the dependencies using requirements.txt file<br>
   <code>pip install -r requirements.txt</code>

3. Finally, start the server<br>
   <code>uvicorn main:app</code>

4. Then visit this url, if you are running the server on localhost<br>
    <code>http://127.0.0.1:8000</code>
